# lagou_crawl

一个含有5万条的16个字段的爬虫。

主要的字段有：
                '路径':url,
                '标题':title,
                '内容':content,
                '薪资':money,
                '地区':district,
                '经验要求':jinyan,
                '学历要求':xueli,
                '全职否':quanzhi,
                '优势':youshi,
                '发布日期':date,
                '公司':gongsi_name,
                '领域':gongsi_lingyu,
                '规模':gongsi_guimo,
                '公司主页':gongsi_web,
                '公司阶段':gongsi_jieduan,
                '公司地址':gongsi_addr
